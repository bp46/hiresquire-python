# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.2.2] - 2026-05-09

### Added
- Native support for AutoGen agents.
- Full `HireSquire` client class with namespace organization.
- Support for calendar and interview scheduling.
- Support for agent key management.
- `Idempotency-Key` support for job creation.

### Fixed
- Fixed version inconsistency in User-Agent header.
- Improved LangChain tool wrapping reliability.

## [1.2.1] - 2026-05-01

### Added
- Initial release with core screening tools.
- LangChain integration support.
- Support for PDF and DOCX parsing.
